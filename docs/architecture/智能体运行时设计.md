# 智能体运行时设计

OpenCode 是本项目的主要执行运行时。后端通过 `opencode serve` 和 OpenCode SDK 控制任务执行。

## 运行流程

1. 创建任务工作区。
2. 准备 OpenCode agent、command、tool 配置。
3. 启动或连接 OpenCode server。
4. 创建 session。
5. 使用指定业务智能体发送用户任务。
6. 订阅 OpenCode 事件流。
7. 将时间线事件和权限请求转发给 Web UI。
8. 持久化输出文件和任务元数据。

## 首个智能体

优先实现 `deep-research`，因为它的流程相对稳定，价值容易验证。
