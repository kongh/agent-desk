# 系统总览

```text
Web UI
  -> API Backend
      -> OpenCode Runner
          -> OpenCode Server
              -> Agent Profiles / Commands / Tools / MCP
          -> Task Workspace
      -> Event Stream
  -> Result Preview
```

## 模块职责

- Web UI：任务输入、实时执行时间线、权限审批、结果预览。
- API Backend：任务状态、OpenCode session 控制、事件转发、工作区生命周期。
- OpenCode Runner：OpenCode server 启停、SDK 调用、session prompt、事件订阅、权限响应。
- Agent Workspace：标准化任务文件系统。
- Business Tools：搜索、资料抽取、报告生成、PPT 大纲、企业连接器等业务工具。
