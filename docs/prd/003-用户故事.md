# 用户故事

## 深度调研

作为用户，我希望输入一个调研主题后，智能体能自动规划、检索、整理证据并生成报告，这样我可以节省资料收集和初稿撰写时间。

作为用户，我希望报告中的关键结论能关联来源，这样我可以验证结果是否可信。

作为用户，我希望能看到智能体正在做什么，这样我可以判断任务是否跑偏。

作为用户，我希望报告生成后可以继续追问和修改，这样我可以逐步得到满意的版本。

## PPT 助手

作为用户，我希望把报告或资料一键整理成 PPT 大纲，这样我可以快速进入演示材料制作阶段。

## 技能生成器

作为用户，我希望把反复出现的业务流程沉淀成可复用技能，这样以后不需要每次重新描述完整步骤。
